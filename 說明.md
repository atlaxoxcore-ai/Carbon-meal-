# 胰島素餐前針助手 — 上線 & 手機使用說明

## 檔案
- index.html — 主程式
- manifest.webmanifest — App 設定
- sw.js — 離線快取
- icon-192.png / icon-512.png — App 圖示

全部要放喺**同一個資料夾**。

## 一、放上 host(需要 HTTPS)
PWA 嘅「加入主畫面」同離線功能，一定要 **HTTPS**。
1. 將成個 `insulin_pwa` 資料夾上載去你個網站，例如 `https://yeelight.hk/sugar/`。
   (SiteGround File Manager 或 FTP 直接拖上去 public_html 下面開個 sugar 資料夾。)
2. 打開 `https://yeelight.hk/sugar/` 確認個 App 載到。

> 本機 double-click 開 file:// 都用到，但離線/安裝/儲存喺部分瀏覽器會唔穩，所以派俾人最好用 HTTPS 連結。

## 二、手機加入主畫面(俾用家)
**iPhone (Safari):** 開連結 → 分享 → 「加入主畫面」→ 完成。之後喺主畫面開，會全螢幕似 App。
**Android (Chrome):** 開連結 → 右上 ⋮ → 「安裝應用程式 / 加入主畫面」。

## 三、資料
- 所有設定、記錄、食物都儲喺**用家自己部手機**(localStorage)，唔會上傳。
- 換機 / 清快取會冇咗 → 叫用家定期喺「記錄」頁「⬇ 匯出備份」。

## 四、派發前建議
- 第一次開會有安全聲明，要撳「我明白」先用。
- 想再加保障，可以喺 index.html 頂部加多一句你公司 / 診所名同免責聲明。

## 五、安全界線(已內置)
- 工具只計數，**唔會自動改用家嘅 ICR/ISF 劑量**。
- 個人比例一定要醫生提供；記錄係畀用家同醫生睇 pattern 去調整。
